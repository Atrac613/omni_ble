# 0.1.0

- Initial public preview release.
- Added a Dart-first BLE API for central and peripheral roles across iOS,
  macOS, Android, Windows, and Linux.
- Added implemented native backends for central GATT workflows and peripheral
  GATT server workflows on every supported desktop and mobile target.
- Added Android runtime BLE permission helpers, rationale helpers, settings
  recovery helpers, and connection tuning APIs.
- Added an example app, device-lab runbook, and validated support matrix for
  cross-platform interoperability testing.
